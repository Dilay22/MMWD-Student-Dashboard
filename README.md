# AcademIQ Student Dashboard

## Project Description
AcademIQ is a student productivity dashboard developed for the Multimedia and Web Design course at International Balkan University.

The project helps students manage academic life through a centralized dashboard that includes:
- Task management
- Weekly schedule planning
- Notes section
- Personalized student dashboard
- Dark mode interface
- Motivational quote widget

The goal of the project is to improve student organization, productivity, and academic planning using modern front-end web technologies.

## How to Run the Project Locally

### 1. Clone the Repository
git clone https://github.com/Dilay22/MMWD-Student-Dashboard

### 2. Open Project Folder
Open the project in Visual Studio Code.

### 3. Run with Live Server
- Right click login.html
- Select "Open with Live Server"

or manually open:
login.html

### 4. Login and Explore
Use the login page to access:
- Dashboard
- Tasks
- Weekly Schedule
- Notes
